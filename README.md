## Installation

```shell
# npm
npm install -g copylen
```

```bash
# yarn
yarn global add copylen
```

```shell
# pnpm
pnpm add -g copylen
```

```shell
# bun
bun install --global copylen
```

## Usage

```shell
copylen <source-file> --len <number-of-characters>

# copylen toolarge.json --len 1000
```

## Options

- `--len, --length, -L`: Number of characters
